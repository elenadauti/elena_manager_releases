# ElenaManager Releases

This repository is only for public ElenaManager desktop release metadata and signed release assets.

Allowed here:
- update-manifest.json
- latest.yml and blockmaps for updater checks
- signed installer and portable .exe assets tracked with Git LFS
- links to signed GitHub Release assets when used
- public certificate files if needed

Never commit:
- .env files
- Supabase keys or passwords
- code-signing .pfx/.p12 files
- backups or client data
- source code from the private app repository
